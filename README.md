
# Ordem Defense

Um jogo **Tower Defense** não comercial e open source inspirado no universo de **Ordem Paranormal** do Cellbit e no jogo Bloons TD 6.

## Sobre o Projeto

Ordem Defense é um jogo de estratégia em tempo real onde você coloca agentes em diferentes mapas para defender contra ondas de criaturas. Gerecie seu dinheiro, saúde e sanidade, faça upgrades e monte sua melhor defesa!

### Estado do Projeto

No momento o projeto esta sendo refatorado para que a gameplay volte a funcionar, mas por enquanto da pra navegar pelos menus e entrar nos mapas. As artes de UI serão refeitas em algum momento, e novos personagens das tempradas recentes serão adicionados. No meu portfolio, fiz uma compilação dos modelos de dentro do jogo: <a target="_blank" href="https://fellipederato.wixsite.com/portfolio#jogos">fellipederato.wixsite.com/portfolio</a>

## Criação

Quase todas as artes, sons, modelos, mapas e scripts foram feitas por mim. pode conter musicas e artes oficiais de Ordem Paranormal nos arquivos, o uso de IA generativa foi utilizada somente de auxílio para escrever os scripts, ja que foi o meu primeiro projeto de programação (não tinha nem o chatgpt na epoca udhsauhdsa). 

## Características

- **Tower Defense Estratégico** - Posicione agentes para derrotar monstros
- **Múltiplos Mapas** - Terrenos variados com desafios diferentes
- **Sistema de Personagens** - Vários agentes com habilidades únicas
- **Upgrades** - Potencialize seus agentes e defesas
- **Efeitos Visuais** - VFX e animações imersivas
- **Sistema de Áudio** - Música, efeitos sonoros e vozes

## Estrutura do Projeto

```
Assets/
├── Animation/          # Controladores e clips de animação
├── Assemblys/          # Scripts dos sistemas principais
├── Clips/              # Áudio (música, efeitos, vozes)
├── Imagens/            # Sprites e texturas
├── Modelos/            # Modelos 3D
├── Scenes/             # Cenas do jogo
├── ScriptsNovos/       # Scripts adicionais
├── Terrenos/           # Terrenos dos mapas
├── VFX/                # Efeitos visuais
└── Materials/          # Materiais
```
